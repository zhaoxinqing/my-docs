## 7、代码块
<!-- 两个 ` 将代码内容包裹起来就是一个行内代码 -->
<!-- 1 个 tab 可以创建一个缩进代码块 -->
<!-- 三个反引号（ ``` ）把需要高亮显示的代码包裹起来，并在第一个 ``` 的末尾标识上语言类型就可以创建一个高亮代码块 -->

```go

type Info struct {
    Name   string `json:"name"`
    Age    int8   `json:"age"`
    Gender int8   `json:"gender"`
}

func main() {
    var info Info
    fmt.Sprintln(info)
}

```
